# Planning Studio Templates

Shared Planning Studio templates live here.

These are operator-facing planning modes, not a domain-specific framework. Use them to move from a vague idea to a buildable plan, then into execution handoff or Knowledge Exchange when the artifact becomes durable.

Recommended flow:
1. project-kickoff
2. discovery-interview
3. scenario-matrix
4. assumptions-challenge-review
5. prd-generator
6. execution-breakdown
7. promotion-to-knowledge-exchange
