# Project Kickoff Template

## When to use
Use at the start of any new project, feature, workflow, tool, or internal initiative.

## Input prompt
```md
Start a Project Kickoff session.

Help me define:
- what we are building
- why it matters
- who it is for
- what success looks like
- what constraints we must respect
- what is explicitly out of scope

Do not write the full spec yet. Ask targeted questions first, then return a clean kickoff brief.
```
